1)Сериализация - это запись объекта в файл
Дессериализация - обратное действие сериализации, то есть поток байтов преобразуется обратно в объект

2)Создание потока
Открытие потока
Запись данных в файл
Закрытие потока

3)Ввод вывод данных
Запись примитивных типов данных
Запись байтов
Запись объектов
Запись символов

4) IO использует потоки для обработки данных, в то время как NIO использует буферы.
IO не имеет такой функциональности и подходит в основном для простых задач ввода-вывода.
